# vim-sendagaya

## Requirements

- Vim 8.0 or later
- [CoreLocationCLI](https://github.com/fulldecent/corelocationcli)
  - `brew install corelocationcli`
- [stdbuf](http://www.gnu.org/software/coreutils/stdbuf)
  - `brew install coreutils`
